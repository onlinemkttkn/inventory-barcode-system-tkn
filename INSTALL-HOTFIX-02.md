# Master 3.4.3 Hotfix-02 — POS Payment V2

## แก้ไข

- ลบช่องรับเงิน ปุ่มเงินด่วน และเงินทอนออกจากหน้าหลัก
- กด `ชำระเงิน` แล้วจึงเปิด Popup
- Quick Cash ใช้ตัวเลขชุดเดียว: 20, 50, 100, 200, 300, 400, 500, 1000
- เพิ่มปุ่ม `เงินพอดี`
- เงินสดเริ่มที่ 0
- เงินรับเป็น 0 หรือน้อยกว่ายอดสุทธิ ยืนยันไม่ได้
- เงินทอนคำนวณแบบ Real-time
- QR/โอน/บัตร ไม่แสดง Quick Cash และไม่คำนวณเงินทอน
- ไม่แก้ SQL

## วิธีติดตั้ง

1. สำรอง `pos.html`, `pos.js`, `pos.css`
2. อัปโหลด 3 ไฟล์นี้ไปที่ root ของ Repository
3. เขียนทับไฟล์เดิม
4. Commit:

`Deploy Master 3.4.3 Hotfix-02 POS Payment V2`

5. รอ Deploy
6. กด `Ctrl + Shift + R`

# Phase 7.4 — Audit Log

ระบบเก็บประวัติการใช้งานสำหรับร้านเถ้าแก่น้อยชลบุรี

## เก็บประวัติ
- เพิ่มและแก้ไขสินค้า
- รับสินค้าเข้า
- เบิก/จ่ายสินค้า
- โอนสินค้าระหว่างสาขา
- รับสินค้าปลายทาง
- เริ่มและปิดรอบตรวจนับ
- ผู้ใช้งาน วันเวลา สาขา และรายละเอียด

## สิทธิ์
- Admin ดู Audit Log ทั้งหมด
- Staff ดูเฉพาะรายการที่ตนเองทำ
- หน้าเว็บไม่มีสิทธิ์แก้ไขหรือลบ Audit Log

## ติดตั้ง
1. รัน `sql/phase07_4.sql`
2. ตรวจสอบ `js/supabase-config.js`
3. เปิด `index.html` ผ่าน Live Server
4. ทดลองสร้าง/แก้ไขรายการ
5. เปิด `audit-log.html`
6. อัปโหลดไฟล์ทั้งหมดทับ Repository เดิม

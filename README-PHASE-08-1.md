# Phase 8.1 — POS ขายหน้าร้าน

สำหรับร้านเถ้าแก่น้อยชลบุรีและสาขา

## ความสามารถ
- เลือกสาขาขาย
- สแกนบาร์โค้ดหรือค้นหาสินค้า
- เพิ่มสินค้าเข้าตะกร้า
- แก้จำนวนและราคาขาย
- ส่วนลดท้ายบิล
- เงินสด / โอน / QR / บัตร / อื่น ๆ
- คำนวณเงินทอน
- ตัดสต๊อกแยกสาขาแบบ Transaction
- ป้องกันขายเกินสต๊อก
- สร้างเลขที่ขายอัตโนมัติ
- ประวัติการขาย
- Audit Log
- Admin ยกเลิกรายการขายผ่าน RPC ได้

## วิธีติดตั้ง
1. รัน `sql/phase08_1.sql`
2. ตรวจสอบ `js/supabase-config.js`
3. เปิด `index.html` ผ่าน Live Server
4. ทดลอง `pos.html`
5. ตรวจสต๊อกใน `branch-stock.html`
6. ดูประวัติใน `sales-history.html`
7. อัปโหลดไฟล์ทั้งหมดทับ Repository เดิม

## หมายเหตุ
ก่อนขาย สินค้าต้องมีสต๊อกใน `branch_inventory` ของสาขาที่เลือก

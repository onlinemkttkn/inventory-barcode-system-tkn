# Phase 8.3 — ระบบสมาชิกและคะแนนสะสม

## ความสามารถ
- สมัครสมาชิกด้วยเบอร์โทร
- เลขสมาชิกอัตโนมัติ
- ค้นหาด้วยชื่อ เบอร์โทร หรือเลขสมาชิก
- เก็บคะแนน ยอดซื้อ และจำนวนครั้งที่ซื้อ
- ผูกสมาชิกกับบิล POS
- สะสมคะแนนอัตโนมัติ
- ใช้คะแนนเป็นส่วนลด
- ประวัติการได้และใช้คะแนน
- ตั้งค่าเริ่มต้น: ยอดซื้อ 100 บาท = 1 คะแนน
- คะแนน 1 คะแนน = ส่วนลด 1 บาท

## วิธีติดตั้ง
1. รัน `sql/phase08_3.sql`
2. ตรวจสอบ `js/supabase-config.js`
3. เปิด `index.html` ผ่าน Live Server
4. ทดลอง `members.html`
5. ขายสินค้าผ่าน POS แล้วเปิด `pos-member.html`
6. ดูประวัติที่ `member-history.html`
7. อัปโหลดไฟล์ทั้งหมดทับ Repository เดิม

## หมายเหตุ
Phase นี้ใช้การผูกสมาชิกกับบิลหลังการขาย เพื่อให้ติดตั้งง่ายและไม่กระทบ POS เดิม
เฟสถัดไปสามารถรวมช่องสมาชิกเข้าในหน้า POS โดยตรงได้

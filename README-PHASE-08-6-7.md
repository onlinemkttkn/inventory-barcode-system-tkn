# Phase 8.6–8.7 — ใบเสร็จ POS และคืนสินค้า

## Phase 8.6
- พิมพ์ใบเสร็จ 58 มม.
- พิมพ์ใบเสร็จ 80 มม.
- รองรับ PeriPage
- พิมพ์ A4
- เลือกจำนวนสำเนา
- แสดงสาขา พนักงาน สมาชิก คะแนน ส่วนลด เงินรับ และเงินทอน
- QR Code เลขที่บิล
- พิมพ์ใบเสร็จย้อนหลังจากประวัติการขาย

## Phase 8.7
- ค้นหาบิลขาย
- คืนสินค้าบางรายการหรือทั้งหมด
- ตรวจจำนวนที่ขาย คืนแล้ว และคืนได้
- ป้องกันคืนเกินจำนวน
- คืนสต๊อกเข้าสาขาอัตโนมัติ
- คำนวณยอดคืนเงิน
- ปรับคะแนนสมาชิกตามสัดส่วนยอดคืน
- เก็บ Audit Log
- จำกัดการคืนสินค้าเฉพาะ Admin
- ประวัติการคืน
- พิมพ์ใบคืนสินค้า

## วิธีติดตั้ง
1. รัน `sql/phase08_6_7.sql`
2. ตรวจสอบ `js/supabase-config.js`
3. เปิดระบบผ่าน Live Server
4. ทดลองใบเสร็จที่ `receipt.html`
5. ทดลองคืนสินค้าที่ `sales-return.html`
6. ดูประวัติที่ `sales-return-history.html`
7. อัปโหลดไฟล์ทั้งหมดทับ Repository เดิม

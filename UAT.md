# UAT

- [ ] ปุ่มเปิดกะแสดงเมื่อยังไม่เปิดกะ
- [ ] ปุ่มปิดกะซ่อนเมื่อยังไม่เปิดกะ
- [ ] ค้นสินค้าและชำระเงินไม่ได้ก่อนเปิดกะ
- [ ] เปิดกะด้วย Employee Code + PIN ได้
- [ ] เปิดกะแล้วปุ่มเปิดกะหาย และปุ่มปิดกะแสดง
- [ ] รีเฟรชหน้าแล้วสถานะกะยังอยู่
- [ ] เลือกสาขาไม่ได้ระหว่างเปิดกะ
- [ ] เงินสด Popup เริ่ม 0 และเงินทอนถูกต้อง
- [ ] ขายหนึ่งครั้งสร้างบิลหนึ่งครั้ง
- [ ] ปิดกะแสดงเงินที่ควรมีและผลต่าง
- [ ] Receipt เปิดและพิมพ์ได้
- [ ] Console ไม่มี Error สีแดงที่เกี่ยวข้อง

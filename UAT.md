# UAT POS Final Stability

- [ ] เปิดระบบโดยไม่มีกะ แสดงหน้ารอพนักงาน
- [ ] เปิดกะใหม่ด้วย Employee Code + PIN
- [ ] ปิดกะแล้วไม่ Logout และกลับหน้ารอพนักงาน
- [ ] มีออเดอร์ค้างแล้วปิดกะไม่ได้
- [ ] ยกเลิกทั้งออเดอร์ต้องใช้ผู้อนุมัติและเหตุผล
- [ ] ผู้ไม่มีสิทธิ์อนุมัติยกเลิกไม่ได้
- [ ] เปิดลิ้นชักนอกการขายต้องกรอกรหัส + PIN + เหตุผลทุกครั้ง
- [ ] เงินสดหลังบันทึกบิลเปิดลิ้นชักหนึ่งครั้ง
- [ ] QR / โอน / บัตรไม่เปิดลิ้นชัก
- [ ] Hardware ล้มแล้วบิลยังสำเร็จ
- [ ] Audit ล้มแล้ว POS ยังใช้งานต่อได้
- [ ] Receipt / VAT / Print ไม่เปลี่ยน
- [ ] Console ไม่มี Error ใหม่

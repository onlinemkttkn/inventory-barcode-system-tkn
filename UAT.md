# UAT Master 3.4.16

## Logout Guard
- [ ] เปิดกะแล้วกดออกจากระบบสีแดง
- [ ] ระบบไม่ Logout
- [ ] แสดงข้อความให้ปิดกะก่อน
- [ ] มีปุ่มเดียว `กลับไป POS`
- [ ] กดกลับไป POS แล้วปุ่มปิดกะได้รับ Focus
- [ ] ปิดกะแล้วกดออกจากระบบได้ตามปกติ

## Responsive
- [ ] เปิดกะ Popup ไม่ล้นหน้าจอ
- [ ] ปิดกะ Popup ไม่ล้นหน้าจอ
- [ ] ยกเลิกออเดอร์ Popup ไม่ล้นหน้าจอ
- [ ] เปิดลิ้นชัก Popup ไม่ล้นหน้าจอ
- [ ] Payment Popup ไม่ล้นหน้าจอ
- [ ] มือถือไม่มี Scroll แนวนอน
- [ ] ปุ่มบนมือถือเต็มความกว้างและกดง่าย

## Regression
- [ ] เงินสดเปิดลิ้นชักตามเดิม
- [ ] QR/โอน/บัตรไม่เปิดลิ้นชัก
- [ ] พิมพ์ใบเสร็จทำงานตามเดิม
- [ ] ปิดกะแล้วแสดงหน้ารอพนักงาน
- [ ] Console ไม่มี Error ใหม่

# UAT

- [ ] ด้านบนขวาไม่มีปุ่มออกจากระบบ
- [ ] Sidebar ยังมีปุ่มออกจากระบบสีแดงด้านล่างซ้าย
- [ ] เปิดกะและปิดกะได้
- [ ] ปิดกะแล้วไม่ Logout และค้างหน้า POS
- [ ] หลังปิดกะ ปุ่มเปิดกะแสดงอีกครั้ง
- [ ] ปิดกะไม่ได้ถ้ายังมีสินค้าในตะกร้า
- [ ] POS รองรับ Desktop / Tablet / Mobile
- [ ] Payment Popup คำนวณเงินทอนถูก
- [ ] ใบเสร็จ 58 มม. / 80 มม. / A4 ไม่ตัดข้อความ
- [ ] QR ใช้ไม่ได้ก็ยังพิมพ์ใบเสร็จได้
- [ ] ปุ่มกลับ POS และประวัติการขายทำงาน
- [ ] Console ไม่มี Error สีแดงที่เกี่ยวกับ POS/Receipt

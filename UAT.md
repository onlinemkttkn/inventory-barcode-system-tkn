## Logout
- [ ] หน้า Dashboard ไม่มีปุ่ม Logout ด้านบน
- [ ] หน้า Reports ไม่มีปุ่ม Logout ด้านบน
- [ ] หน้า Scanner ไม่มีปุ่ม Logout ด้านบน
- [ ] หน้า Print Labels ไม่มีปุ่ม Logout ด้านบน
- [ ] หน้า Generator ไม่มีปุ่ม Logout ด้านบน
- [ ] ทุกหน้ายังมีปุ่ม Logout สีแดงใน Sidebar

## Mobile Camera
- [ ] Scanner เปิดกล้องหลังบนมือถือ
- [ ] Receive มีปุ่มสแกนกล้องและเพิ่มสินค้าที่พบได้
- [ ] Issue เปิดกล้องและเพิ่มสินค้าได้
- [ ] Transfer เปิดกล้องและเพิ่มสินค้าได้
- [ ] iPhone/Safari ใช้ ZXing fallback ได้
- [ ] Android/Chrome สแกน Barcode/QR ได้
- [ ] ปิด Dialog แล้วไฟกล้องดับ
- [ ] ปฏิเสธสิทธิ์กล้องแล้วมีข้อความชัดเจน
- [ ] ไม่มี JavaScript Error สีแดง

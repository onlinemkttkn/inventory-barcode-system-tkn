# TKN POS / ERP — Final Stable Core v1.2

แพ็กนี้แก้ชื่อไฟล์และการอ้างอิงให้ตรงกับไฟล์ที่ระบบใช้งานจริง

## ไฟล์ที่ต้องอัปโหลดทับ

อัปโหลด 3 ไฟล์นี้ไว้ที่ root ของ Repository:

```text
dashboard.html
pos.html
pos.js
```

ต้องเลือกเขียนทับไฟล์เดิมทั้งหมด

## สิ่งที่แก้ถาวร

### POS

- `pos.html` โหลด `./pos.js?v=1.2.0` โดยตรง
- ไม่มีความสับสนระหว่าง `pos.js` กับ `pos-fixed.js`
- ช่องรับเงินตามยอดสุทธิอัตโนมัติ
- เปลี่ยนจำนวนสินค้า ราคา หรือส่วนลดแล้วรับเงินอัปเดตทันที
- เมื่อกรอกยอดเงินจริงเอง ระบบจะไม่เขียนทับ
- เงินทอนคำนวณแบบ Real-time
- วิธีชำระเงินที่ไม่ใช่เงินสดใช้ยอดสุทธิในการบันทึก

### Dashboard

- ปุ่ม Dashboard ในหน้า POS ชี้ตรงไป `./dashboard.html`
- `dashboard.html` เป็นหน้า Unified ที่รวม Phase 08.2, 09.2 และ 9.5

## ขั้นตอนติดตั้ง

1. สำรองไฟล์เดิม หากต้องการ
2. อัปโหลด `dashboard.html`, `pos.html`, `pos.js`
3. Commit:

```text
Deploy final stable POS core v1.2
```

4. รอ GitHub Pages 1–3 นาที
5. เปิด:

```text
https://onlinemkttkn.github.io/inventory-barcode-system-tkn/pos.html
```

6. กด `Ctrl + Shift + R`

## ทดสอบครั้งสุดท้าย

1. เพิ่มสินค้า 3 ชิ้น ราคา 150
2. ยอดสุทธิและช่องรับเงินต้องเป็น 450
3. กรอกรับเงิน 500 เงินทอนต้องเป็น 50
4. เปลี่ยนส่วนลดเป็น 50 ยอดสุทธิต้องเป็น 400
5. หากยังไม่ได้แก้ยอดรับเงินจริง ช่องรับเงินต้องตามเป็น 400
6. กด Dashboard ต้องกลับ `dashboard.html`

## ไม่ต้องอัปโหลด

ไม่จำเป็นต้องใช้ `pos-fixed.js` อีกหลังติดตั้งชุดนี้

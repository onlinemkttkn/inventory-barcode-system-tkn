# Phase 9.5 Navigation Fix v1.1

## ปัญหาที่แก้

หน้าเดิมบางหน้าพึ่งพา:

```js
history.back()
```

เมื่อเปิดผ่าน Popup หรือกดข้ามหลายหน้า ประวัติเบราว์เซอร์จึงพาย้อนผิดหน้า

## เส้นทางใหม่แบบตายตัว

```text
Dashboard 9.5
  → ค้นหาบิล
    → คืนสินค้า (Popup)
      → ปิดกลับค้นหาบิล

Dashboard 9.5
  → ประวัติคืน
    → ใบคืนสินค้า
      → ปิดกลับประวัติคืน

Dashboard 9.5
  → รายงานคืน
    → กลับ Dashboard 9.5
```

## ติดตั้ง

1. อัปโหลดทุกไฟล์ใน ZIP โดยรักษาโครงสร้าง
2. ไม่ลบไฟล์เดิม
3. Commit:

```text
Phase 9.5 - Fix deterministic navigation v1.1
```

4. เปิดหน้าใหม่:

```text
dashboard-integrated-phase-9-5-v1-1.html
```

5. กด `Ctrl + Shift + R`

# POS Payment + Dashboard Hotfix v1.1

## แก้ปัญหา

1. เมื่อเพิ่มสินค้า เปลี่ยนจำนวน เปลี่ยนราคา หรือใส่ส่วนลด:
   - ช่อง `รับเงิน` จะอัปเดตตาม `ยอดสุทธิ` อัตโนมัติ
   - `เงินทอน` จะอัปเดตทันที

2. เมื่อพนักงานพิมพ์ยอดรับเงินจริงเอง:
   - ระบบจะหยุดเติมอัตโนมัติ
   - คำนวณเงินทอนจากยอดที่พิมพ์

3. เมื่อเปลี่ยนวิธีชำระเงินกลับเป็นเงินสด:
   - ระบบเติมยอดสุทธิให้อัตโนมัติอีกครั้ง

4. ปุ่ม Dashboard ในหน้า POS:
   - กลับไป `dashboard.html` ตัว Unified
   - ไม่ย้อนกลับ Dashboard เวอร์ชันเก่า

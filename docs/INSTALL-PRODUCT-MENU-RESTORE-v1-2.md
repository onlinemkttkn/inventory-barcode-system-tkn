# Product Menu Restore v1.2

## สิ่งที่แก้

เพิ่มเมนูกลับเข้า Dashboard:

- จัดการสินค้า → `products-admin.html`
- จัดการสต็อกสินค้า → `product-stock-admin.html`

ไม่มีการแก้ SQL, JavaScript, Supabase หรือไฟล์ Phase 9.2

## ติดตั้งแบบง่าย

1. อัปโหลดไฟล์นี้เข้า root ของ Repository:

```text
dashboard-integrated-phase-9-5-v1-2.html
```

2. Commit:

```text
Restore product management menus v1.2
```

3. เปิด:

```text
https://onlinemkttkn.github.io/inventory-barcode-system-tkn/dashboard-integrated-phase-9-5-v1-2.html
```

4. กด `Ctrl + Shift + R`

## ตรวจสอบ

- กด `จัดการสต็อกสินค้า` ต้องเปิด `product-stock-admin.html`
- เมนู Phase 9.2 และ Navigation เดิมต้องทำงานเหมือนเดิม

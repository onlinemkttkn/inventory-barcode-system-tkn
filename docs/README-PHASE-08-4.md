# Phase 8.4 — Product Management

## ความสามารถ
- รายการสินค้า
- เพิ่มสินค้าใหม่
- แก้ไขสินค้า
- สร้างบาร์โค้ดอัตโนมัติ
- หมวดหมู่หลักและหมวดหมู่ย่อย
- หน่วยนับ
- ยี่ห้อสินค้า
- ราคาทุน ราคาขาย VAT และสต๊อกขั้นต่ำ
- รูปสินค้าแบบ URL
- สต๊อกเริ่มต้นแยกสาขา
- ปรับสต๊อกแยกสาขา
- เชื่อม Audit Log
- จำกัดเพิ่ม/แก้ไข/ปรับสต๊อกเฉพาะ Admin

## วิธีติดตั้ง
1. รัน `sql/phase08_4.sql`
2. ตรวจสอบ `js/supabase-config.js`
3. เปิด `index.html` ผ่าน Live Server
4. เปิด `products-admin.html`
5. เพิ่มหมวดหมู่ หน่วย และยี่ห้อที่ `categories-admin.html`
6. ปรับสต๊อกที่ `product-stock-admin.html`
7. อัปโหลดไฟล์ทั้งหมดทับ Repository เดิม

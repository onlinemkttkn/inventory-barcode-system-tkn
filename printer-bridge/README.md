# Rongta 80mm Cash Drawer Bridge

1. ติดตั้ง Driver และตั้งชื่อเครื่องเป็น `RONGTA 80mm Series Printer`
2. ต่อสายลิ้นชักเข้าพอร์ต Drawer ของเครื่อง Rongta
3. คลิกขวา `START-RONGTA-BRIDGE.bat` แล้ว Run as administrator
4. เปิด `http://127.0.0.1:17890/health` ต้องเห็น `ok:true`
5. เปิด Bridge ค้างไว้ระหว่างใช้งาน POS

คำสั่ง Drawer ใช้ ESC/POS: `ESC p 0 25 250`
